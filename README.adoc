== Mode

* `N` = Normal
* `I` = Insert
* `C` = Command
* `V` = Visual
* `VL` = Visual Line
* `VB` = Visual Block
* `T` = Terminal

== Special Key

[source, shell]
----
nvim -c ":h key-notation"
----
